# Machine Learning Projects

This repository showcases multiple machine learning projects demonstrating data analysis, feature engineering, and predictive modeling on real-world datasets.

## Projects

### Titanic Survival Prediction
- Classification model using passenger data
- Built using Python and Scikit-learn
- Focused on predicting survival outcomes based on features such as age, class, and gender
- 📂 [View Project](titanic-project/)

### Netflix Data Analysis & Predictive Classification
- End-to-end machine learning project involving data cleaning, exploratory data analysis, and feature engineering.
- Built classification models (Decision Tree, Logistic Regression)
- Achieved ~71% accuracy in classifying content as Movies or TV Shows
- 📂 [View Project](netflix-project/)
